ShiftManager Application API documentation version v1

https://api.rostergrid.com/v1

Introduction

This API is created to interact with the rostergrid workplace web application and rostergrid employees application.

  • Request
  • Response

URI Parameters

  • userId: required(string)

Query Parameters

  • workplace: required(integer)

    Id of the workplace to retrieve shifts from

    Example:

    1
  • upcoming: required(boolean)

    Specify whether to retrieve only upcoming shifts, putting true to this value means only upcoming shifts are retrieved.

    Example:

    true
  • month: (integer)

    Specify which month's shifts to retrieve

    Example:

    9
  • year: (integer)

    Specify which year's shift to retrieve

    Example:

    2019
